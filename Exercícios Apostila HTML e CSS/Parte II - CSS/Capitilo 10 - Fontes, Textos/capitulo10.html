Exercícios Capítulo 10 - Fontes, Textos


10.3. Alterar o tamanho da fonte de todos os títulos de nível 3 de uma página HTML para 2rem

        
    h3 {
        font-size: 2rem;
    }

10.1. Seletor CSS para modificar a cor do texto de todos os parágrafos de uma página HTML

        
    p {
        color: #ff3399;
    }

10.2. Defina “Roboto” como a fonte principal e “Arial” como a segunda opção caso “Roboto” não esteja disponível.

        
    body {
        font-family: "Roboto", "Arial", sans-serif;
    }
Seletor CSS que aplique o estilo desejado a um elemento <span> da classe destaque. 
        
    span.destaque {
        font-weight: bold;
    }
Seletor para aplicar um estilo de fonte em itálico para citações textuais contidas em elementos <blockquote>
        
    blockquote {
        font-style: italic;
    }
