Exercícios Capítulo 11 - Pseudo-seletores


Utilize o pseudoseletor ::before para adicionar um asterisco (*) antes dos títulos de nível 1. (11.1)

        
    h1::before {
        content: "* ";
    }

Aplique o pseudoseletor ::after para adicionar uma linha horizontal após todos os parágrafos. (11.2)

        
    p::after {
        content: "";
        display: block;
        border-bottom: 1px solid #000;
        margin-top: 8px;
    }

Utilize o pseudoseletor ::first-letter para aumentar a fonte da primeira letra dos parágrafos. (11.3)

        
    p::first-letter {
        font-size: 2em;
        font-weight: bold;
    }

Aplique o pseudoseletor ::first-line para alterar a cor da primeira linha de todos os parágrafos. (11.4)

        
    p::first-line {
        color: #ff3399;
    }
